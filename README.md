Snoozeec2
=========

This is the EC2 system service for Snooze. This service allow to interact with Snooze through an EC2 like API.
Only a subset of methods of the huge EC2 API is supported.

## Supported Actions

`DescribeInstances` 

`DescribeImages`

`RunInstances`

`TerminateInstances`

`RebootInstances`
